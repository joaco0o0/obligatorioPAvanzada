Trivia Jeopardy Mario Carminatti, Joaquin Heredero
Instrucciones:
Abrir la carpeta desde VS Code
Una vez abierta, ir al main y darle a run
En la terminal aparecerán 2 opciones. Iniciar o Salir
Si se inicia, el juego le muestra 5 preguntas al azar al usuario, de las cuales elige una, dando input entre 1 y 5.
Para esa pregunta aparecerán 3 opciones que el usuario selecciona con un input entre 1 y 3. 
Al finalizar la quinta pregunta, al usuario se le devuelve su puntaje total y el tiempo que tardó.
En caso de que quiera reintentar o un nuevo jugador quiere jugar para compararse con el anterior, se puede presionar Iniciar juego para volver a contestar las 5 preguntas. Sino simplemente se presiona salir.
